# Ro'eh — רֹאֶה

> Celui qui voit ce que les autres ne voient pas.

Ro'eh est un outil de transparence numérique causale, open source, collaboratif et souverain.

## Ce qu'il fait
- Analyse tout ce qui entre et sort de la machine
- Surveille les modifications intrasystème (malwares, ransomwares)
- Explique la causalité — l'action qui a déclenché chaque flux
- Intercepte avant les actions dangereuses
- Propose un mode narratif pour tous et un mode expert pour les analystes

## État du projet
Maquette d'interface — design et architecture en cours.
